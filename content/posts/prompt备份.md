## 模拟面试的 Prompt

以下是关于模拟面试的 Prompt 的一些建议：

1.设置面试情景 Prompt

可以给出一个具体的面试场景，如"你正在参加一家科技公司的产品经理面试"。这样可以让被面试者有明确的背景和角色定位

2.提供职位描述 Prompt

给出被面试的职位的主要职责和要求，让被面试者了解自己需要展现的关键技能和经验

3.设置面试官 Prompt

可以扮演不同性格的面试官，如严厉、友善等，让被面试者体验不同风格的面试官

4.设置问题 Prompt

准备一系列常见的面试问题，如自我介绍、工作经历、未来规划等

引导被面试者回答这些问题，展现自己的能力

5.反馈 Prompt

在模拟面试过程中，给予被面试者及时的反馈和点评帮助他们了解自己的表现，并提出改进建议

6.情景变化 Prompt

可以设置一些意外情况，如被问到意外问题、遇到冲突等

考验被面试者的应变能力和临场反应

总的来说，设置丰富的 Prompt 可以让模拟面试更加贴近真实情况，帮助被面试者更好地准备和练习。同时也要注意 Prompt 的灵活性，让整个过程更具互动性和挑战性。
